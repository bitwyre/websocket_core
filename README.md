# Websocket Core (Rust)

![Build Status](https://github.com/bitwyre/websocket_core/workflows/Build/badge.svg)

Websocket generic server library for:

1. Periodic message broadcast
2. Eventual (Pubsub) message broadcast
3. Async request reply
